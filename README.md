## ğŸ“Œ Deskripsi Sistem

Zona Laut terdiri dari dua peran utama:
- **Admin:** Mengelola data kapal, pengguna, dan tangkapan.
- **Nelayan:** Menginput data tangkapan dan memantau posisi kapal.

Sistem ini dilengkapi dengan:
- **Integrasi API BMKG:** Menampilkan kondisi cuaca terkini.
- **WhatsApp Gateway:** Verifikasi OTP saat login.
- **Peta Interaktif:** Menampilkan posisi kapal dan zona penangkapan.
